# Visual Studio Code integration

Glass is designed to work seamlessly within [Visual Studio Code](https://code.visualstudio.com). The [Glass extension](https://marketplace.visualstudio.com/items?itemName=foundation.vscode-glass) can be easily installed from the Visual Studio Code Marketplace. {{ className: 'lead' }} {{className: 'lead'}}

## VSCode settings

<Row>
  <Col>
    <Properties>
      <Property name="glass.defaultModel" type="string">
        Default model for prompts. Defaults to `gpt-3.5-turbo`.
      </Property>
      <Property name="glass.defaultLanguageMode" type="string">
        The language to transpile to. Defaults to `TypeScript`.
      </Property>
      <Property name="glass.openaiKey" type="string">

        Your Anthropic API key. You can create one on your [Anthropic dashboard](https://console.anthropic.com/account/keys).

## Commands

The Glass extension provides a number of commands to help you work with Glass files.

### `Glass: Transpile all`

This command transpiles all `.glass` files in your workspace and writes the output to the directory of your choice which you can [configure in your VSCode settings](/vscode).

### `Glass: Transpile current file`

Similar to the `Transpile all` command, but only transpiles the current file.

### `Glass: Open Playground`

This command will open and run the current file in the Glass playground sidebar.

### `Glass: Open Settings (UI)`

Shortcut to open the Glass settings in your `settings.json` file.

### `Glass: Show Glass Output`

This shortcut shows the logs of your playground

### `Glass: Open Documentation`

Shortcut to open the Glass documentation in your browser.

### `Glass: Open support chat`

Coming soon! Shortcut to open the Glass support chat in your browser.
