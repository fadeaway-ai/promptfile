# Installing the Glass VSCode Extension

Glass is designed to work seamlessly within Visual Studio Code. Go here to install the Glass [extension](https://marketplace.visualstudio.com/items?itemName=foundation.vscode-glass).

## Accessing Glass Settings
Open the Settings by opening the VSCode Command Palette by pressing `Command (⌘) + Shift (⇧) + P`. Run the `Glass: open settings` command.
