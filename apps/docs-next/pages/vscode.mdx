# Visual Studio Code integration

Glass is designed to work seamlessly within [Visual Studio Code](https://code.visualstudio.com). The [Glass extension](https://marketplace.visualstudio.com/items?itemName=foundation.vscode-glass) can be easily installed from the Visual Studio Code Marketplace.

## Settings

<br />

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Type</th>
      <th>Description</th>
      <th>Default</th>
    </tr>
  </thead>
  <tbody>
    {Object.entries(vscodeSettings.contributes.configuration[0].properties).map(([name, val], i) => (
      <tr key={i}>
        <td>
          <code>{name}</code>
        </td>
        <td>{val.type}</td>

    ))}
  </tbody>
</table>
